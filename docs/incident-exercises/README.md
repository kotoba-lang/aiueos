# Incident Exercises

This directory stores repo-local security exercise evidence for aiueos.

The records are intentionally lightweight and EDN-friendly: each exercise names
the scenario, inputs, expected signals, observed gaps, and closure state. They
are not a substitute for production monitoring evidence, but they keep the
incident-response playbooks executable during development.

- `compromised-signer-tabletop.md`
- `unauthorized-key-release-tabletop.md`
- `host-capability-denial-alert-simulation.md`
